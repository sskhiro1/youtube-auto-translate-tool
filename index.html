YT YouTube Auto Translator

Chrome extension

YouTubeの字幕翻訳を、リンクから一気に。

YouTubeリンクと翻訳先を選ぶだけで、動画ページを開き、字幕、設定、Auto-translate、言語選択まで自動で進めるツールです。Google Cloud Translation NMTの公式対応リストに合わせ、194言語候補に拡張しました。

拡張機能ZIPをダウンロード 入れ方を見る
  1. 1Chromeで chrome://extensions/ を開く
  2. 2デベロッパーモードをオンにする
  3. 3ZIPを展開し、extensionフォルダを読み込む
